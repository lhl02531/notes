# 产品定位和产品差异化

# 产品定位

Q: 如何确定产品定位

A: 套用一个句式，向什么人群（用户），在什么场景下（使用场景），通过什么方法（使用方式）， 提供了什么服务(服务),解决了他们的需求（用户需求），带来什么价值（商业利益）

抖音为例， 所有人，所有场景，通过短视频为主的提供内容服务，解决了观看者的娱乐需求，博主的创造需求，商家的推广需求，博主可以通过短视频收获经济利益，商家可以快速性价比高的推广商品，从而平台可以从中抽收佣金。

# 差异化

Q: 如何找到产品差异化

A:

  1. 产品: 功能，设计， 使用体验。栗子:抖音在短视频的竖屏设计，下滑刷新新视频是一个很简单容易上瘾的操作，而且大数据推荐也在记住用户的喜欢的短视频类型，这样用户刷视频的用户体验会很好。
  2. 服务: 个性化，情感化，制造惊喜。 网易云音乐曾在年末推出一年的听歌总结，这样用户可以通过网易云音乐的这份报告，唤起自己的个人的情感，这当中的很大一部分用户会自发转发到朋友圈推广，这样的定制服务也是推广的很好示例。海底捞通过服务员的热烈服务，生日的时候也会给消费者通过唱歌等活动来庆祝生日，同时又敏锐的把握到单人用餐的尴尬，会有陪伴人偶在消费者对面起到陪伴效果。
  3. 渠道， 腾讯视频号可以借助微信体系进行发展
  4. 品牌，品牌的宣传口号，标识，比如微信的标识，现在只要出现就知道腾讯微信，淘宝百度，
