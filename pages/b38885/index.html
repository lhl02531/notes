商业价值理解

# 商业价值理解

# 商业化设计前要回答的三个问题

1.客户是谁？who 2.能给客户带来什么价值？value 3.为什么？why

# 栗子

以百度为例

who, 广告主 value, 找到潜在用户 value, 大量且精准的搜索推荐引擎

# 做商业化设计关注的四个指标

  1. LTV(Life time Value): 用户生命周期价值
  2. CAC(Custom Acquisition Cost):用户获取成本
  3. PBP(Payback Period):投资回收期
  4. ROI(Return On Investment):投资回报率,分子是收益，分母是成本

# 常见的商业模式

前向收费 后向收费
收费对象 c 端用户 b 端用户，流量型 b端用户，服务依托型
收费方式 阶梯定价 按流量效果定价 按服务效果定价
提供的服务 区分于其他用户的权益 广告曝光和转化 业务支撑
提供的形式 会员服务
增值服务
内容付费
软件付费		 横幅广告
信息流广告
搜索竞价排名
佣金抽成		 技术服务
售后支持

前向付费的经典例子有，视频，小说，游戏，付费软件等
后向收费， 流量型，抖音广告，b站广告 服务型，阿里云服务等
