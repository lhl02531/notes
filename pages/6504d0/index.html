# 用户思维

# 为什么强调用户思维

产品诞生始就需要有用户。你当然可以随意去设计产品，但是这样随意的产品并不会有用户，有市场接受。
合格的产品不是自嗨出来的，你需要跟用户共情，找到用户需求，这样的产品才会被用户接受。
合格的产品使用体验应该是顺风顺水的，而不是逆水行舟的感觉。设想你的产品的某个功能需要用户思考，让用户违反他的正常使用体验，这些都不符合用户使用体验，从而影响到他对产品的判断。

# 一些很好的例子

# 视频播放网站的例子

爱奇艺, 腾讯会有只追某个明星的片段的功能，满足了一些追星粉可能只想反复看片段但是又不想操作的需求。
b站的弹幕上线了人脸防弹功能，但你识别到人脸时，弹幕不会挡住人脸。
优酷上线甄嬛传一键回宫功能，满足了剧迷想只想刷某些集数，如果没有这个功能，就要剧迷手动调集数然后再手动调进度的麻烦，
这些其实都是非常好的优化，如果你深入到用户中，完全没办法知道视频网站还需要做这些功能。

# 尝试分析一下日常产品

  1. 用户属性： B站的用户主要集中在18-35岁之间，其中以学生和上班族为主。男性用户占比稍高，但女性用户的数量也在逐渐增加。用户主要分布在一二线城市，具有较高的教育程度和收入水平。

  2. 用户偏好： B站用户对动画、漫画、游戏、科技等领域的内容较为感兴趣。他们喜欢观看UP主创作的原创视频和转载的优质内容，其中有些热门UP主拥有庞大的粉丝群体。用户在B站上还喜欢参与弹幕互动，讨论热门话题和二次元文化。

  3. 用户行为： B站用户有较强的观看习惯，每天登录B站已经成为他们的习惯。用户喜欢在平台上观看长视频、追新番、订阅UP主、评论互动等。同时，B站也是一个购物和直播平台，用户可以通过购买UP主推荐的商品和打赏支持他们。

  4. 用户诉求： B站用户的主要诉求是学习和娱乐。他们希望通过观看优质视频来获取知识、提高技能，同时也享受二次元文化和弹幕互动带来的娱乐体验。用户的诉求优先级因个人偏好而异，有些用户更注重学习，而有些用户则更注重娱乐。

  5. 用户心理特征： B站用户具有年轻化、有活力的特点，他们追求新鲜感和创新意识。同时，用户也有一定的社交需求，希望能够通过B站的弹幕和评论互动与其他用户建立联系，形成自己的社群。
